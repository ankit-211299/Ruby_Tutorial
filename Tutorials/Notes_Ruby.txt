Data Types in Ruby:-
1. Strings
2. Integers
3. Floating Points
4. Boolean
5. nil - nothing/empty but still a value.

Numbers In Ruby:

1. Add - +
2. Sub - -
3. Mult - *
4. Div - /
5. Pow - ** or ^

.to_s - converts number to Strings
1) .abs() - gives absolute value
2) .ceil() - upper value round off
3) .floor() - lower value round off
4) .round(n) - n decides the number of decimal places

Math function:
1. Math.pow()
2. Math.sqrt()
3. Math.log()

Arrays in Ruby:

arr = Array[.,.,.,.,]; #syntax
we can store string,number,boolean in an array.
we can mix store in the array

1. arr.length() - gives the length of the array
2. arr.include? "element" - gives booblean output according to the presence of element in the array
3. arr.reverse() - reverse the order of the element in the array
4. arr.sort() - to sort the array alphabetically
